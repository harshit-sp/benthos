This processor is designed to convert messages with key-value pairs, using any specified delimiter, into a structured JSON representation.

Introduced in version 4.25.

```yml
# Config fields, showing default values
label: ""
kv:
  pair_delimiter: "" # No default (required)
  key_value_separator: "" # No default (required)
  target_field: "" # No default (optional)
```

## Fields

### `pair_delimiter`

Regex pattern to use for splitting key-value pairs.


Type: `string`  

### `key_value_separator`

Regex pattern to use for splitting the key from the value within a key-value pairs.


Type: `string`  

### `target_field`

The field to insert the extracted keys into.


Type: `string`
